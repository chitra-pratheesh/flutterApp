## Spring Boot Flutter Login Registration


`Git Clone`
```
git clone https://github.com/chitra-pratheesh/flutterApp.git
```
`Change Directory`
```
cd flutterApp
```

`Change Directory`
```
cd api
```
`Run Command`
```
mvn spring-boot:run
```

`Change Directory`
```
cd flutterApp
```

`Run Command`
```
flutter run -d chrome 
```

`Or Open VSCode and`
```
Ctrl/Cmd + F5
```



`Url for POST API`

http://localhost:8080/login

http://localhost:8080/register
